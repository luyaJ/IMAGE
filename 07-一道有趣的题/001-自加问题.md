今天晚上做的 完美世界 的笔试题，遇到了这样一道题。

```js
var a = 0, b = 2;
a = b+++5*++b;
```

好多`+`，我懵逼了。赶紧百度一下，但是出来这样一道题：
```js
int a = 5, b;
b = (++a) + (++a);
```

这里的操作相当于：
(1) ++a;
(2) ++a;
(3) b = a + a;

上面两句运行完后，a的值肯定是7，因为自增了两次，所以最后的值是`b = a + a = 7 + 7 == 14`。

所以我遇到的笔试题也是一样的：
```js
a = (b++) + 5 * (++b) 
```

(1) `b++`：b++ = 2，b = 3
(2) `++b`：++b = 4

所以，`a = 2 + 5 * 4 = 22`，答案就是 22 了。
